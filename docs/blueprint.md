# **App Name**: CASE CareerConnect

## Core Features:

- Student Profile: Students can create profiles with academic details, skills, portfolio links, CV upload, and video introductions.
- Company Portal: Companies can create profiles, manage their booth assignments, download student shortlists, and manage interviews.
- AI Job Matching: AI-powered job matching based on CGPA, skills, and preferences.
- Interview Scheduling: Students can book interviews with calendar sync and receive reminders.
- Volunteer Management: System to register and manage volunteers with role assignments, shift scheduling, and attendance tracking.
- Real-time Analytics Dashboard: Dashboard providing insights into total registered students, companies, interview success rates, and department-wise placements.
- Resume Analysis: AI-powered tool to analyze student resumes and offer improvement suggestions. The tool reasons about the current content, and might decide that changes are needed.

## Style Guidelines:

- Primary color: SSCIET Green (#008000) for brand recognition and trustworthiness.
- Background color: Light green (#E0F2E0), a very desaturated hue near the primary color.
- Accent color: Forest green (#228B22) for calls to action and highlights.
- Body and headline font: 'Inter', a sans-serif for a modern and neutral look.
- Use clear, professional icons to represent different features and roles.
- Modern dashboard UI with a clean navigation sidebar and mobile-first design.
- Subtle animations and transitions for a smooth user experience.
